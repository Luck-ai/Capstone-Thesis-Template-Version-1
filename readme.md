## Prerequisites
- Install [Miktex](https://miktex.org/download])
- Install [TexStudio](https://texstudio.org/#download)
- Clone the repository

## Setup
- Open Tex Stduio and navigate to **Options** tab
- Click on Configure Tex Studio
- In the build tab, change the default compiler to **LuaLaTeX**
- *Optional* in the **Internal PDF Viewer** tab, enable the auto recompile option to compile whenever any chnage is made automatically

- Click the build and view button to compile and view the pdf
